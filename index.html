Simon say GAME

Press any key to start the game

Rules to Play the Game -

It is a simple game which checks the remembring power in you.

  1. Press any key to start the game.
  2. The player have to enter the color that blinks.
  3. If the user enters correct, the level will be Incremented.
  4. Now another color will blink, The player have to remember the first color blinked in first level.
  5. The user have to enter the sequence from the starting

    6. For Example,

    • In Level 1, the red color blinks
    • The user Clicks the red color (i.e. Correct)
    • Now in Level 2, the yellow color blinks,
    • The user have to enter the sequence as (red, yellow)
    • Then only the user can go on to Level 3

Enjoy The Game.....
